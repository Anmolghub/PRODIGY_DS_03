# PRODIGY_DS_03

# Customer Purchase Prediction Using Decision Tree Classifier

# Overview

This project is part of the Prodigy InfoTech Data Science Internship.

The objective of this project is to build a Decision Tree Classifier that predicts whether a customer will subscribe to a bank term deposit based on demographic and campaign-related information.

## Dataset

The dataset used in this project is the Bank Marketing Dataset, which contains information collected from direct marketing campaigns conducted by a banking institution.

Dataset Source:

https://www.kaggle.com/datasets/janiobachmann/bank-marketing-dataset

## Technologies Used

## Features Used

Some important features in the dataset include:

- Age
- Job
- Marital Status
- Education
- Balance
- Housing Loan
- Personal Loan
- Contact Type
- Campaign Information

### Target Variable

- **Yes** → Customer subscribed to the term deposit
- **No** → Customer did not subscribe

## Model Evaluation

The Decision Tree Classifier was evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- Feature Importance Analysis

## Key Insights

- Most customers did not subscribe to the term deposit.
- Customer demographics and previous campaign information influence subscription decisions.
- Certain features contribute more significantly to prediction outcomes.
- Decision Trees provide interpretable and easy-to-understand classification results.
